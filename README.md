Customize GitHub's `<relative-time>` elements with your preferred date and time formats.

自定义 GitHub `<relative-time>` 元素的日期与时间显示格式。

## Configuration / 配置项

You can access these options via the Userscript menu:
你可以通过脚本管理器的菜单访问以下配置：

- `DISPLAY_FORMAT`: Format for the visible timestamp (Default: `YY-MM-DD HH:mm`).
- `TOOLTIP_FORMAT`: Format for the hover tooltip (Default: `YYYY-MM-DD HH:mm:ss`).
- `DEBUG_LOG`: Enable verbose logging in the console for troubleshooting (Default: `false`).
